# namecard

> Simplistic Nuxt name card

## Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# lint tool
$ npm run lint
$ npm run lintfix

```
* Note that lint will run before commiting.
