# namecard

This simplistic Nuxt namecard is shown on [the homepage of my personal website](https://poyea.me/).

## Build Setup

```bash
# install dependencies
$ npm install

# Requires Node.js 20+ (use `nvm install 20 && nvm use 20`)

# serve with hot reload at localhost:3000
$ npm run dev

# generate static site for production (outputs to dist/)
$ npm run generate

# or build for server deployment
$ npm run build
$ npm run start
```

### Linting

```bash
$ npm run lint
$ npm run lintfix
```
